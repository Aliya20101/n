Weather DiaryАвтор: Алия Фахртдинова

Описание:Приложение «Дневник погоды» позволяет фиксировать ежедневные погодные условия: дату, температуру, описание и наличие осадков. Данные сохраняются в файл JSON. Доступна фильтрация по дате и температуре.

Как запустить:

Установите Python 3.8+.
Клонируйте репозиторий.
Запустите weather_diary.py.
Используйте интерфейс для добавления и просмотра записей.
Примеры использования:

Добавьте запись: 2026-05-06 | 18 | Ясно | нет.
Отфильтруйте записи за 2026-05-06.
Покажите все записи с температурой выше +10°C.
Данные сохраняются автоматически при добавлении новой записи.
